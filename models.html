Models

A model is any class with a default constructor that has accessible getters and setters for the fields that are going to interact with the view.

public class Note {
    private long time;
    private int id;
    private String title;
    private String text;


    public long getTime() {
        return time;
    }

    public void setTime(long time) {
        this.time = time;
    }

    public int getId() {
        return id;
    }

    public void setId(int id) {
        this.id = id;
    }

    public String getTitle() {
        return title;
    }

    public void setTitle(String title) {
        this.title = title;
    }

    public String getText() {
        return text;
    }

    public void setText(String text) {
        this.text = text;
    }
}

The model is marked as a NgModel inside of the scope.

// Declare your scope
@NgScope
public class NoteFragment extends Fragment {
    // declare your models within your scope
    @NgModel
    Note note;
}

Any models in a scope will be insantiated when the scope is attached to a view if it is null or it is not an instance of Model

Models can then be used in the attribute scripts.

<EditText
    android:id="@+id/et_title"
    ngAndroid:ngModel="note.title"
    .../>

<TextView
    android:id="@+id/tv_time"
    ngAndroid:ngText="getTimeString(note.time)"
    .../>

<EditText
    android:id="@+id/et_text"
    ngAndroid:ngModel="note.text"
    .../>

// other ways they can be used
"functionName(model.boolValue ? 'this string' : 'bool value was false')"
"model.a + model.b"

You can check the scripts page for more information on the scripts.

Models can also be declared with interfaces

public interface Note {
    long getTime();
    void setTime(long time);
    int getId();
    void setId(int id);
    String getTitle();
    void setTitle(String title);
    String getText();
    void setText(String text);
}

That model declared as an interface would be identical to the Note class in the NgAndroid framework.

The interface would be implemented at compile time.
